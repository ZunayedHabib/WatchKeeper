Tips for a healthy setup
🌍Create your Supabase project in the region closest to you. Choose the region closest to your location for the best response times.
⏱Free tier projects pause after 7 days of inactivity. The first request after a pause takes 3–8 seconds — this is normal. Just wait it out and it will be fast afterward.
🔑Use the Publishable Key (starts with sb_publishable_) from the API Keys tab — not the legacy anon key.
🗄️If tables are missing, go to your Supabase project → SQL Editor → run the SQL from the Guide tab in WatchKeeper.
🔁If a new project feels slow on first connect, wait 10 seconds and try again. A brand new project needs one warm-up request.
